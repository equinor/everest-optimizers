### Installation

1.  (optional) Create and activate a virtual environment:

    ```bash
    uv venv
    source .venv/bin/activate
    ```

2.  Install the project:

    ```bash
    pip install .
    ```

### Running the project

To run the optimization test, execute the following command:

```bash
python test_optimization.py
```
